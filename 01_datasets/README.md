# Dataset Information

This folder contains information related to the AdventureWorksDW2025 dataset used in this SQL portfolio project.

## Dataset
AdventureWorksDW2025

## Description
AdventureWorksDW2025 is a data warehouse dataset representing a fictional retail business, structured using a star schema model with fact and dimension tables.

Key tables include:
- Fact tables: FactInternetSales
- Dimension tables: DimCustomer, DimGeography, DimSalesTerritory, DimProductCategory, DimProductSubcategory, DimProduct.

The dataset enables analysis across sales transactions, customer profiles, product categories, geographic regions, and time.

## Purpose
This dataset is used to analyze:
- Business trends over time
- Customer behavior
- Product and category performance
- Product profitability
- Geographic sales distribution
